\# MR 排球訓練系統（MR Volleyball Training System）



\## 一、專案簡介

本專案為一套「混合實境（Mixed Reality, MR）排球訓練系統」，使用 Meta Quest 3 搭配 Unity 開發。

系統透過即時手部追蹤與物理模擬，讓使用者能在真實空間中練習排球的基本動作，例如接球、托球與扣殺。



本專案以「技術穿刺（Technical Spike）」為目標，驗證 MR 手部互動、動作辨識與排球物理模擬的可行性。



---



\## 二、系統功能重點

\- 即時手部追蹤（Hand Tracking）

\- 手部揮動速度計算與動作辨識

\- 排球物理模擬（彈性、速度、方向）

\- 混合實境場景建構（Passthrough + Anchor）

\- NPC 練習模式（接球 / 托球 / 扣殺）
